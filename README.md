# KAAJ KAAM - A Remote Job Portal for Bangladesh

A modern remote job portal designed specifically for job seekers in Bangladesh.

## 🔗 Live Demo

- **Frontend**: https://kaajkaam.netlify.app/
- **Backend**: https://kaaj-kaam-a-remote-job-portal-for-9etv.onrender.com

> **Note**: The backend is hosted on Render's free tier. Free instances spin down with inactivity, which can delay requests by 50 seconds or more.

## 📝 About

KAAJ KAAM is a dedicated job portal platform that connects remote job opportunities with professionals in Bangladesh, making it easier to find and apply for global remote positions.
